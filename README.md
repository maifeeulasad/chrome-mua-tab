# chrome-mua-tab
A minimalist Chrome extension that enhances your new tab experience with beautiful wallpapers and quotes.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/maifeeulasad/chrome-mua-tab.git
   ```

2. Load in Chrome:
   - Open Chrome/Edge/Brave or any Chromium-based browser
   - Navigate to `chrome://extensions/`
   - Enable "Developer mode" in the top-right corner
   - Click "Load unpacked" in the top-left
   - Select the cloned `chrome-mua-tab` directory

## Updating

Simply run `git pull` in the extension directory and refresh the extension in `chrome://extensions/`
